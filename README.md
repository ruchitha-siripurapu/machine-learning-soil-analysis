# 🌾 Machine Learning Soil Analysis

A Django-based web application that uses **machine learning** to analyze soil properties and predict soil fertility.  
The project helps farmers and researchers understand soil quality and make better agricultural decisions.

---

## 🚀 Features

- 🧠 Predicts soil fertility using machine learning models
- 📊 Allows users to upload soil datasets (CSV format)
- 🌍 Displays prediction results through an interactive web interface
- ⚙️ Built with Django for smooth web integration
- 💾 Uses SQLite as the default database

---

## 🛠️ Tech Stack

- **Python 3**
- **Django**
- **Scikit-learn / Pandas / NumPy**
- **HTML / CSS / Bootstrap**
- **SQLite**
